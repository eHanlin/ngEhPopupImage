ngEhPopupImage
===================================

## Install

```bash
bower install ngEhPopupImage
```

## Install node modules

```bash
npm install
```

## Dependencies

* angular
* jquery
* animate.css
* bootstrap (optional)

## Usage

#### html
```html
<img src="img.jpg" ng-eh-popup-image="img.jpg" width="250" alt="" />
```

#### js
```js
app = angular.module("app",["eHanlin"]);
```

## DEMO

```bash
gulp server
```

## Build

```bash
gulp
```
